# Daily check-in scripts

This repository stores independently deployable daily check-in projects for authorized website and app accounts.

| Target | Project | Deployment |
| --- | --- | --- |
| NodeSeek | [`sites/nodeseek/`](sites/nodeseek/) | QingLong |
| NovalPie | [`sites/novalpie/`](sites/novalpie/) | QingLong |

Each site directory contains its own configuration example, security guidance, offline tests, and deployment documentation. Credentials must only be supplied through the target platform's protected environment-variable store.
